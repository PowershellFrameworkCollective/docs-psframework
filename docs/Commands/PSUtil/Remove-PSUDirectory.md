# Remove-PSUDirectory

## SYNOPSIS
Removes the current directory and moves the console to the parent folder.

## SYNTAX

```
Remove-PSUDirectory [-Force] [-EnableException] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes the current directory and moves the console to the parent folder.

## EXAMPLES

### EXAMPLE 1
```
rcd
```

Tries to remove the directory and step back one level.
